¿Qué es Steemeditor?

Es una aplicación web desarrollada para los usuarios de Steemit. Permite acomodar el diseño de sus publicaciones (post) mediante el control de los botones en la barra de herramientas, sin necesidad de escribir o tener conocimiento de códigos de estilos de páginas web.

Ver más características

Beneficios

Fácil arreglo del post
Corrector ortográfico
Control del diseño seleccionando el texto
Posibilidad de publicar el post desde el editor
Mejora la apariencia de los post en menor tiempo
Posibilidad de guardar el post en un archivo de texto
Fácil etiquetado

Limitaciones

  • No se podrá cargar las imágenes directamente desde steemeditor, es necesario subirlas a otra página web que permita el alojamiento de imágenes, por ejemplo Facebook, Pinterest, Mega.nz, google drive , one drive...

Precio

  • Para usuarios de steemit con más de 20 steem power, tienen 15 días de uso gratis, si desea continuar con el editor el costo es 0.5 STD

  • Para usuarios de steemit con menos de 20 steem power, tienen 2 meses de uso gratis, si desea continuar con el editor el costo es 0.15 STD

El depósito se realizará una sóla vez para adquirir la licencia de uso ilimitada.

El descuento para los usuarios con menos de 20 steem power es para que reciban un apoyo en su comienzo en steemit.
